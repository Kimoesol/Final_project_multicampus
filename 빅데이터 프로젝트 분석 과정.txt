주제 선정 -> 데이터 수집 -> 전처리 및 저장 -> 데이터 분석 / 모델링 -> 모델 검증을 통한 최적화

-> 데이터 마트 생성 -> 성능 검증 -> 모델 검증 -> 데이터 분석 / 모델링 -> 전처리 -> 데이터 탐색 -> 운영 시스템 적용 -> 모니터링 -> 유지 보수
